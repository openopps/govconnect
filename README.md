# govconnect

This is a fork of the original which is at: https://github.com/18F/govconnect

# Requirements

- Ruby (2.2.3 recommended)
- Bundler gem
- [Pandoc](http://pandoc.org/installing.html) (try `brew install pandoc` on OS X)

# Usage

- `bundle install`
- `jekyll serve`
- `open http://127.0.0.1/govconnect`
